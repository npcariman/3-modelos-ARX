1 - Modelo de persistencia¶

1.1 Descripción¶

El primero modelo que se utilizará será el modelo de persistencia definido como

$$\hat{y}_t = y_{t+1}$$

Este modelo simplemente proyecta el ultimo valor de la salida hacia el futuro.

# Modelo de persistencia
def persistence_model(x):
    '''
    Se considera que la entrada x es un vector fila, por lo que simplemente se devuelve
    el ultimo valor del vector
    '''
    x = x[-1]
    return x

1.4.1 Calculo de $V_N$¶

$$V_N = \cfrac{1}{N} \sum_{t=1}^{N} l [L(q)\varepsilon],$$

con:

  • Filtro $L(q)$: $$L(q) = 1$$
  • Norma $l(\vec{\varepsilon})$ $$l(\varepsilon) = \frac{1}{2} \vec{\varepsilon}^T \cdot I \cdot \vec{\varepsilon} = \frac{1}{2} \sum_{i=1}^6 \varepsilon_i^2$$
In [9]:
# Definimos las funciones para la evaluacion de los errores

# filtro L(q):
def L(x):
    return x

# Norma l(epsilon):
def l(x):
    return 0.5 * sum(x ** 2)

def V_N(df_error):
    global L, l
    df_error = copy.copy(df_error)
    df_error = df_error.apply(L, axis=0)
    df_error = df_error.apply(l, axis=1)
    return sum(df_error) / len(df_error)
